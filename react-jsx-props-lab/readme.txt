# React JSX & Props Lab

## 📌 Overview
This project demonstrates the basics of React, including:
- Creating functional components
- Using JSX
- Importing and rendering components
- Passing and displaying props
- Reusing components with different data

---

## 🚀 Project Setup

### 1. Create the Project (Vite)
```bash
npm create vite@latest react-jsx-props-lab
cd react-jsx-props-lab
npm install
npm run dev
````

---

## 📂 Folder Structure

```
react-jsx-props-lab/
│
├── src/
│   ├── components/
│   │   ├── Welcome.jsx
│   │   └── UserCard.jsx
│   ├── App.jsx
│   └── main.jsx
```

---

## 🧩 Features Implemented

### ✅ Functional Component

* `Welcome.jsx` displays a simple greeting using JSX.

### ✅ Component Rendering

* `Welcome` component is imported and displayed in `App.jsx`.

### ✅ Props Usage

* `UserCard.jsx` receives `name` and `email` as props.
* Displays dynamic content based on props.

### ✅ Reusable Components

* `UserCard` is used multiple times with different values.
